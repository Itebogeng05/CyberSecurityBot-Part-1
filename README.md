# Cybersecurity Awareness Bot (CSAB)

## Overview
The Cybersecurity Awareness Bot (CSAB) is a C# console-based chatbot designed to educate users about basic cybersecurity practices. It provides interactive guidance on topics such as password safety, phishing, malware, and safe browsing.

This application aims to promote safer online behavior, especially for everyday users who may not be familiar with common cyber threats.

---

## Features
- Voice greeting on startup  
- Console-based chatbot interface  
- Personalized interaction using the user's name  
- Typing effect for realistic chatbot responses  
- Multiple cybersecurity topics:
  - Password safety  
  - Phishing and scams  
  - Safe browsing  
  - Malware protection  
  - Social engineering  
  - Two-Factor Authentication (2FA)  
- Exit commands (`exit`, `quit`, `bye`, `done`)  

---

## Technologies Used
- C#
- .NET Console Application
- System.Media (for audio playback)
- System.Threading (for typing effect)
